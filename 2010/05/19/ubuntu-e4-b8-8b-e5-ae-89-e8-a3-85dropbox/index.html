ubuntu下安装dropbox

2010.05.19 Ivan 工作相关
在两个实验室间同步文档和保存，dropbox果然是最好的选择，今天把ubuntu的笔记本装上了dropbox，过程曲折，记录为下
两种途径：
1、网站下载 https://www.getdropbox.com/downloading?os=lnx   不过我虽然能打开网址看到下载地址，在伟大的教育网中却下不来，此方式适合外网人士。下载后安装，安装后，打开终端，输入：killall nautilus 回车，搞定。
2、加源
首先 gksudo gedit /etc/apt/sources.list
然后加入以下
deb http://linux.getdropbox.com/ubuntu karmic main deb-src http://linux.getdropbox.com/ubuntu karmic main
保存退出后，再装一个GPG的Key
gpg `--`keyserver pgp.mit.edu `--`recv-keys 3565780
刷新一下 sudo apt-get update
开始安装 sudo apt-get install nautilus-dropbox
安完之后我出现了以下问题：
In order to use Dropbox, you must download the proprietary daemon.
程序在此卡住，后来经上网多方研究发现卡住是因为程序要下载，但下载源被墙了，唉
好在网上有有心人，自己下了一个，挂在了如下地址：
http://freedomhui.com/wp-content/uploads/2010/05/dropbox-lnx.x86-0.7.111.tar.gz
下载完后是个压缩包，~/.dropbox-dist/，记住是将解压之后的结果放在这路径下。从源码可知，第一次启动Dropbox时，他会在这个路径下查找~/.dropbox-dist/dropboxd程序，就是在/home/用户/这个路径下，不过.dropbox-dist是隐藏文件，要注意
然后，就正常启动了，伟大的dropbox
