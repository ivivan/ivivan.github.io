制作 chrome web app

2010.07.13 Ivan 工作相关

每次听豆瓣电台都要打开一个网页，很是让人烦，chrome的扩展功能又不全，官方又没有解决办法 。无奈下，研究了一下做了个豆瓣电台的app，试用一下效果良好 ，就是图标丑了一点，实在懒得找好看的

步骤：

  • 找一个不错的图标，需要2个尺寸：24x24和128x128
  • 按以下形式写一个manifest.json文件，具体详细的语法规则见google官方文档，直接从三个官方里复制一个，然后用笔记本打开修改一下就可
